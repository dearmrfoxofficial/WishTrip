Trip Wishlist (Lite)

Define flexible travel criteria and get periodic updates via GitHub Issues. Free. No backend.

Create a Trip

After generating, copy the JSON and save it into your repo under /trips/your-trip.json.
The GitHub Action will read all files in /trips and post updates to Issues you can subscribe to.

Trip JSON

If you later add a TEQUILA_API_KEY secret, the updates will include live prices & apply your max price threshold.
